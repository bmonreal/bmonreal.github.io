Whereas Kamiokande and IMB are big monolithic water tanks, BUST is modular. Each module is a 60-liter tank of scintillator with one PMT. BUST is a three-story building with four layers of these tanks and a TON of #zillowgonewild potential. 0 br, 0 ba, 3180 detector modules
