Kamiokande's data went to a PDP/11-60, then to magnetic tape, via a tiny I/O buffer. Detector sends data to buffer memory, buffer memory goes to tape. If new data comes in when the buffer was already full, it just ... vanishes. The buffer might be full! Why?

For example, the TAPE had a WRITE ERROR, which would prompt it to pause, rewind (6250 bytes per inch on 2400' reels), and retry. Oyama recalls that 10-second retries happened "about once an hour"; less often if the write heads were kept extra clean.
